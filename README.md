# cheetoz-hook (Rust + ImGui demo)

Ten projekt pokazuje prosty interfejs ImGui w Rust:
- przełączanie menu klawiszem `Insert`,
- tytuł okna `cheetozhook`,
- slider `CPS` w zakresie 1-30,
- checkbox `Autoclicker` i tekst statusu.

> Uwaga: to jest **samodzielna aplikacja desktopowa** do nauki UI.
> Nie zawiera kodu do wstrzykiwania `.so` do procesów ani omijania zabezpieczeń.

## Uruchomienie

```bash
cargo run
```
